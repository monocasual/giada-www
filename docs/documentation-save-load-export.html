User guide
Save and load projects.

Giada stores songs as projects. A project is a folder named (project name).gprj that contains all the samples used in your current song, along with a JSON-based file called (project name).gptc that stores the project data. You can save and load a project from the File menu in the main menu bar.

Because a project is just a regular folder in your file system, you can browse it, add new content, and even edit the project file if something goes wrong, although ideally that should never be necessary. You can also create the project file by hand or generate it with a script. Be careful not to delete any files from the folder, or you may end up with missing samples.

A project is portable: you can save it on one computer and move it to another. Note, however, that audio plug-ins are not stored inside the project, so make sure the required plug-ins are also installed on the target system.
